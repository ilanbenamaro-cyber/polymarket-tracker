Methodology

How these numbers are computed
  • Probabilities = Polymarket midpoint prices for each "above $X" market (P(YES)).
  • Implied median = the valuation where the cumulative probability P(cap > X) crosses 50%, by linear interpolation between the two bracketing thresholds.
  • Implied mean (approx) = bucket-weighted expected value: each bucket's probability mass × its midpoint. Tail assumptions: the below-lowest bucket uses midpoint (lowest − $0.15T); the above-highest bucket uses (highest + $0.40T).
  • 50% confidence band = the 25th–75th percentile valuations of the implied distribution (where P(cap > X) crosses 0.75 and 0.25).
  • Bucket % = P(> this threshold) − P(> next threshold); the density chart adds a "<$1T" bar so all buckets sum to ~100%.
  • History sourced from the Polymarket CLOB price-history API at daily fidelity; early dates have fewer thresholds (markets were added over time).
